# Security Policy

## Supported Versions

| Version    | Supported          |
| ---------- | ------------------ |
| >= 3.10.16 | :white_check_mark: |
| < 3.10.16  | :x:                |

## Reporting a Vulnerability

This project has an automated workflow to check vulnerabilities and security flaws but
if you want to report anything, please open an issue in the project to be validated.
